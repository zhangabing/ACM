#  实验一进程调度
### 多级反馈队列

初步设定三级队列，第一级队列设置时间片1， 第二级为2，第三级为4
### 可变时间片轮转

每一轮修改一次时间片，当前的时间片设置为等待序列之中的还需时间片的均值
# 实验二作业调度
### 单道批处理之先来先服务

按提交时间排序直接运行作业，并计算周转时间和带权周转时间

### 多到批处理之短作业优先

内存限制100K，磁带机限制5台

1. 按提交时间排序，到达提交时间后放入准备序列
2. 在资源运许的条件下，从等待序列中挑选最短作业优先进入运行序列
3. 在运行队列中挑选已完成作业进入已完成队列

# 实验三存储管理

### 首次适应算法

根据申请的所需内存，从头到尾，找到可以存放当前作业的内存区即放进去

释放内存则是找到作业所在内存区，释放之后判断前后合并空闲内存问题

### 最佳适应算法

根据申请的所需内存，从头到尾搜索一遍，找到众多可以存放该作业的内存中选取最小的内存区将本作业存储

释放内存则是找到作业所在内存区，释放之后判断前后合并空闲内存问题
